## `Document`

- chapter
- section
- paragraph
- sentence

## Ambiguities

- they are language-agnostic
- same pronunciation, different spelling; e.g. عصیر / اسیر
- same spelling, different pronunciation; e.g. ملک
- same pronunciation, same spelling, different meaning; e.g. روم
